## 项目全局依赖
* 包管理工具: pnpm、yarn
* 命令行工具: node、npx
* 文档工具: jsdoc、jdsoc2md、conventional-changelog、docsify
* 格式校验工具: eslint、prettier
* 打包工具: rollup

## 特殊命令
* husky初始化: npx husky-init && npm install 
* 加husky: npx husky add .husky/commit-msg 'npx --no-install commitlint --edit "$1"'
